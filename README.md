# Мой Аниме Список

Веб-приложение для отслеживания просмотренных аниме.

## Функции
- Регистрация и авторизация пользователей
- Добавление аниме в список
- Отслеживание количества эпизодов и времени просмотра
- Управление списком аниме

## Технологии
- Python 3.9
- Flask
- SQLAlchemy
- PostgreSQL
- HTML/CSS
